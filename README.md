# ستارلايت: ما دون النجوم

لعبة ويب ثابتة بأسلوب **Pixel Art** مبنية باستخدام **Phaser 3** وتعمل مباشرة من المتصفح عبر `index.html` بدون npm وبدون أي خطوة بناء.

## القصة

تسير الشخصية في رحلة طويلة ظنّت فيها أنها تقاتل العالم، ثم تكتشف أن أصعب باب هو الباب الذي تحرسه من الداخل.
تواجه في الطريق عصابة ستارلايت، حيث يمثل كل بوس جزءًا داخليًا: الغضب، الكبرياء، الخوف، الندم، الإنكار، ثم الذات الأخيرة.

## طريقة التشغيل (محليًا)

1. افتح المستودع.
2. شغّل خادم ملفات ثابتة بسيط **أو** افتح `index.html` مباشرة في المتصفح.
3. تبدأ اللعبة دون تثبيت أي حزم.

> ملاحظة: اللعبة لا تحتاج npm إطلاقًا.

## النشر على GitHub Pages

1. ارفع المشروع إلى مستودع GitHub.
2. من إعدادات المستودع افتح **Pages**.
3. اختر المصدر: الفرع الرئيسي (`main` أو `master`) والمجلد الجذري (`/root`).
4. احفظ الإعدادات.
5. ستعمل اللعبة مباشرة لأن نقطة الدخول هي `index.html`.

## التحكم

- الفأرة: اختيار الأزرار والقوائم.
- `Enter`: متابعة الحوارات.
- `Esc`: فتح/إغلاق قائمة الإيقاف داخل الساحة.
- الحركة في الساحة: الأسهم أو `A` و `D`.
- هجوم أساسي: `J`
- هجوم ثانوي: `K`
- اندفاع: `L`

## هيكل الملفات

- `index.html`: نقطة الدخول وتحميل Phaser من CDN.
- `styles.css`: تنسيق الصفحة وواجهة RTL.
- `src/main.js`: تشغيل اللعبة.
- `src/config.js`: إعدادات Phaser وربط المشاهد.
- `src/state.js`: الحالة العامة والحفظ المحلي.
- `src/scenes/`: جميع مشاهد اللعبة.
- `src/entities/`: اللاعب، الأعداء، البوسات، المقذوفات.
- `src/systems/`: الأنظمة (الصوت، السينما، الحوارات، الجولات، الترتيب المحلي...).
- `src/data/`: بيانات القصة، الحوارات، الأعداء، الجولات.
- `src/ui/`: مكونات الواجهة مثل HUD وصندوق الحوار.
- `assets/`: توثيق الأصول وسياسة الاستخدام.

## التقنيات

- HTML + CSS + JavaScript
- Phaser 3 من CDN:
  - `https://cdn.jsdelivr.net/npm/phaser@3.80.1/dist/phaser.min.js`
- WebAudio للموسيقى والمؤثرات
- `localStorage` للحفظ والترتيب المحلي
